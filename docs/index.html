Velnix Editor is a lightweight native text editor for Linux developers. Built with GTK, Scintilla, and Lexilla, it focuses on fast startup, low resource usage, and practical editing workflows for text and source code.

Philosophy

Velnix Editor is not trying to become a heavy IDE. It focuses on the daily editing loop: open quickly, edit comfortably, search confidently, save safely, and stay out of the way.

The goal is to provide a middle ground between minimal editors and full development environments: lightweight but capable, native but modern, practical without unnecessary complexity.
